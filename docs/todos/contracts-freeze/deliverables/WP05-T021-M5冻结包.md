## 6. Design -> Build 映射（含三件套）

| Design 结论 | Build 目标 | 映射说明 |
|---|---|---|
| M5 gate 单入口 | scripts/ci/wp05_contract_gate.sh | 统一 configure/build/registration/test/full-suite 流程 |
| M5 发布可调用门禁 | tests/CMakeLists.txt 新增 dasall_wp05_contract_gate | 复用 dasall_contract_tests，避免重复编译链 |
| V1 Ready 收口 | required tests 包含 V1ReadyChecklistContractTest | 把评审结论转为可执行门禁 |
| 冻结阻断策略 | 脚本非零返回码 | 缺注册、测试失败、全量失败任一触发阻断 |

### Build 三件套

| 项 | 内容 |
|---|---|
| 代码目标 | scripts/ci/wp05_contract_gate.sh；tests/CMakeLists.txt |
| 测试目标 | 复用 WP05 20 个 contract tests（含正向与负向门禁验证） |
| 验收命令 | bash scripts/ci/wp05_contract_gate.sh |

### 阻塞项与解阻条件

| 阻塞项 | 证据 | 解阻条件 |
|---|---|---|
| BLK-A：required tests 与 CMake 注册清单不一致 | ctest -N -L contract 缺失测试名 | 修复 tests/contract/CMakeLists.txt 注册后重跑 gate |
